# Privacy Policy

Last updated: 2026-03-11

## Overview

*hetu - Hide SSN on Finnish authentication sites* is a browser extension that hides visible Finnish personal identity code values on specific login pages using local CSS rules.

## Data collection

This extension does not collect, store, transmit, or sell personal data.

Specifically, the extension does NOT:

- collect browsing history
- capture form input
- read or send authentication data
- use analytics or tracking tools
- send data to external servers

## Permissions and access

The extension applies CSS only to explicitly matched URLs listed in `manifest.json`.

No additional permissions are requested.

## Data sharing

No user data is shared with any third parties.

## Security

The extension contains static files only and does not load remote code.

## Changes to this policy

If this policy changes, updates will be published in this repository and reflected in extension release notes.

## Contact

For questions, open an issue in the project repository.
